# 🌿 FloraIntellect

Proyecto para ExpoKinal 2026.

## Descripción
FloraIntellect es una plataforma que ayuda a identificar plantas y proporcionar información mediante inteligencia artificial.

## Características
- Identificación de plantas
- Información botánica
- Interfaz amigable
- Sistema de consultas

## Tecnologías
- HTML
- CSS
- JavaScript
- Node.js

## Instalación

npm install
npm start

## Equipo
- Derek González

## Objetivo

Ayudar a estudiantes, agricultores y entusiastas de la botánica a obtener información sobre plantas mediante herramientas tecnológicas e inteligencia artificial.
